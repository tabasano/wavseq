# Smml

simple Music Macro Language to MIDI

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'smml'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install smml

## Usage

    m=MmlTracks.new
    m.compile('infile.mml','outfile.mid')
